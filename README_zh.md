# 沐光之行

## 项目简介

一款 2D 平台跳跃游戏。玩家将操控角色，通过各种平台跳跃挑战并克服障碍。

---

## 开发环境

* **游戏引擎：** Unity
* **编程语言：** C#
* **运行平台：** Windows

本项目基于 Unity 游戏引擎开发，专注于轻量化 2D 游戏玩法以及受中国神话启发的视觉设计。

---

## 快速开始

### 运行项目

#### 使用 Unity 打开

克隆仓库：

```bash
git clone https://github.com/z54413210-star/Chasing-the-Sun.git
```

1. 使用 Unity Hub 打开项目
2. 在 Unity 中加载项目
3. 打开主场景并点击 `Play` 运行游戏

#### 通过网页链接游玩

进入：

[https://chasing-the-sun.netlify.app/](https://chasing-the-sun.netlify.app/)

等待加载完成后即可开始体验游戏！

你也可以在初始场景中自由选择关卡进行游玩，不过更推荐从第一关开始，因为游戏会提供一些基础教学，帮助你熟悉操作方式。

---

## 游戏玩法

* 使用键盘控制角色移动与跳跃
* 躲避障碍并前往终点
* 通过平台跳跃关卡推进游戏进度
* 按下 `E` 键可在“过去”与“现在”之间切换

  * 现在场景会更加昏暗
  * 过去场景则更加明亮
