# Chasing the Sun

A 2D platformer game. Players control our character and overcome obstacles through various platforming challenges.

## Development Environment

* **Game Engine:** Unity
* **Programming Language:** C#
* **Platform:** Windows

The project was developed with the Unity game engine and focuses on lightweight 2D gameplay and myth-inspired visual design.

## Getting Started

### Run the Project

#### Open by Unity engine

1. Clone the repository:

```bash
git clone https://github.com/z54413210-star/Chasing-the-Sun.git
```

2. Open the project with Unity Hub.
3. Load the project in Unity.
4. Open the main scene and click **Play**.

#### Open by link
Enter https://chasing-the-sun.netlify.app/, wait for some time and enjoy our game!

You can also choose levels to play in the first scene.
But it is better to start from the beginning, as there will be some instructions to help you get familiar with the game control.

### Gameplay

* Use the keyboard to control movement and jumping.
* Avoid obstacles and move toward the destination.
* Complete platforming sections to progress through the game.
* Press "E" to switch between past and present! Present scene is darker while the past one is brighter.
